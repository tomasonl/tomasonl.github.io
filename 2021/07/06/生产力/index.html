IDEA 快捷键

  • try/catch快捷键
    Ctrl + Alt + t
  • 格式化代码
    去掉空白： Ctrl + Shift + J
    格式化代码： Ctrl + Alt + L
  • 大小写切换
    Ctrl+Shift+u
  • 优化依赖引入Imports
    Ctrl+Alt+O
  • 查找/替换
    双击Shift 全局查找
    Ctrl+Shift+F 目录内查找
    Ctrl+R 替换
    Ctrl+Shift+R 目录内替换
